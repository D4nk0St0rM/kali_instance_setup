## pip2

```
wget https://bootstrap.pypa.io/pip/2.7/get-pip.py
chmod u+x get-pip.py
sudo python2 get-pip.py
sudo rm get-pip.py
```

## pip3

```
wget https://bootstrap.pypa.io/get-pip.py
chmod u+x get-pip.py
sudo python3 get-pip.py
sudo rm get-pip.py
```
